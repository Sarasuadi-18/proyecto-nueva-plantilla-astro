## Ejemplo de Solicitud

A continuación, se muestra un ejemplo de cómo enviar una solicitud al endpoint:

<Tabs>
  <TabItem label="Laravel php">
    ```php
    // Tu código de Laravel aquí
    $response = Http::withHeaders([
        'Accept' => 'application/json',
    ])->post('https://api-sandbox.factus.com.co/oauth/token', [
        'grant_type' => 'refresh_token',
    ]);
    ```
  </TabItem>

  <TabItem label="Node js">
    ```javascript
    // Tu código de Node.js aquí
    const response = await fetch('https://api-sandbox.factus.com.co/oauth/token', {
      method: 'POST',
      headers: { 'Accept': 'application/json' },
      body: new URLSearchParams({ 'grant_type': 'refresh_token' })
    });
    ```
  </TabItem>

  <TabItem label="Curl">
    ```bash
    curl -X POST https://api-sandbox.factus.com.co/oauth/token \
      -H "Accept: application/json" \
      -d "grant_type=refresh_token" \
      -d "client_id=tu_id" \
      -d "client_secret=tu_secret" \
      -d "refresh_token=tu_token"
    ```
  </TabItem>
</Tabs>






### Parámetros del Cuerpo (Body)

| Parámetro | Tipo | Descripción | Requerido |
| :--- | :--- | :--- | :---: |
| `grant_type` | `string` | Tipo de concesión: `refresh_token` | ✅ |
| `client_id` | `string` | ID del cliente proporcionado | ✅ |
| `client_secret` | `string` | Secreto del cliente proporcionado | ✅ |
| `refresh_token` | `string` | El token de refresco previo | ✅ |
